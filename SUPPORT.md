# Support

Thanks for using **Petshop**! If you need help, here's how to reach us.

## Support channels

- **GitHub issues** — the fastest way to report a bug or request a feature. Open an issue on the repository's issue tracker with steps to reproduce and what you expected to happen.
- **Email** — for questions that aren't a good fit for a public issue (private data, security concerns), email the maintainers at **support@petshop.example**.

## Before you file

- Check the [README](./README.md) for setup and the API endpoint reference.
- Check [CLAUDE.md](./CLAUDE.md) for conventions and the full command reference.

## QA & testing questions

For questions about running the test suites (lint, unit, and end-to-end) or verifying a change locally, see [TESTING.md](./TESTING.md).
